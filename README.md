# Hotel Khoai - Website

This is the repository for the Hotel Khoai website, a premium web application representing Hotel Khoai located in the Khoai Valley, Tripura.

## Features
- **Home Page**: Introduces the heritage, core stats, and signature dishes with visual media.
- **Our Story**: Explains the history, milestones (2016-2024), and core values.
- **Menu**: Displays the restaurant's authentic Bengali menu offerings.
- **Gallery**: Showcases beautiful images of our spaces (The Bamboo Hall, Fairy Lit Gazebo, and more).
- **Find Us**: Contact info, opening hours, and location details with directions.

## Running the Code

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
